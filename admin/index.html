Admin — Content Guide

This site is modular and content-driven. Products are JSON files; adding one automatically updates the catalog, the Products mega menu filters, and the product detail page.

Add a product

  1. Create a JSON file under content/products/ (e.g., my-build.json).
  2. Add the filename to content/products/index.json in the items array.
  3. Visit products.html. Use the “Products ▾” menu to filter (Gaming, Professional, Premium, Budget, Featured).

Schema (copy/paste and edit):

{
  "title": "Build Name",
  "short": "One-line summary.",
  "description": "Longer description with optional line breaks.",
  "image": "assets/your-image.jpg",
  "category": "game", // or "professional"
  "badges": ["premium", "silent", "RTX 4080"],
  "services": ["Diagnostics", "Upgrade Planning"],
  "featured": false
}

“Premium”/“Budget” filters match words in badges (case-insensitive). “Featured” reads featured: true.
